## 1. 도입 배경 (Why Blockchain for Rural Rest)

### 1.1. 현재 플랫폼의 한계와 블록체인이 해결하는 문제

Rural Rest는 한국 농촌 빈집을 리모델링한 숙소를 글로벌 여행자에게 연결하는 플랫폼이다. 현재 다음과 같은 구조적 한계가 존재한다:

| 한계 | 블록체인 솔루션 |
|------|---------------|
| 빈집 리모델링 자금 조달의 어려움 | RWA 토큰화로 글로벌 소액 투자 유치 |
| 해외 게스트의 높은 결제 수수료 (카드 3-5%) | Solana 결제로 수수료 0.01% 미만 |
| 숙박 경험의 일회성 소비 | NFT 기반 경험 증명 + 재방문 인센티브 |
| 플랫폼 의존적 리뷰 신뢰도 | 온체인 트랜잭션으로 검증 가능한 숙박 이력 |

### 1.2. 빈집 재생 + 블록체인 시너지

농촌 빈집 재생은 사회적 가치(지역 활성화)와 경제적 가치(임대 수익)를 동시에 갖는다. 블록체인은 이를 투명하게 측정하고 글로벌 자본을 연결하는 인프라 역할을 한다:

- **투명한 소유권**: 토큰 기반 분할 소유로 빈집 재생 과정이 공개적으로 추적 가능
- **글로벌 접근성**: 지리적 제약 없이 누구나 한국 농촌 재생에 참여 가능
- **자동화된 수익 분배**: 스마트 컨트랙트로 임대 수익이 투자자에게 자동 배당

### 1.3. Solana 체인 선택 이유

| 기준 | Solana | Ethereum | Polygon |
|------|--------|----------|---------|
| TPS | ~65,000 | ~15 | ~7,000 |
| 트랜잭션 수수료 | ~$0.00025 | ~$2-20 | ~$0.01 |
| 확인 시간 | ~400ms | ~12초 | ~2초 |
| NFT 생태계 | Metaplex (성숙) | OpenSea (최대) | 제한적 |
| Compressed NFT | 지원 (cNFT) | 미지원 | 미지원 |
| 한국 개발자 생태계 | 성장 중 | 가장 큼 | 중간 |

**선택 근거**: 숙박 완료 시 자동 NFT 민팅, 마이크로 결제, 대량 포인트 트랜잭션 등 Rural Rest의 유스케이스는 **저비용 + 고처리량**이 핵심이다. Compressed NFT를 활용하면 NFT 1개 민팅 비용이 $0.0001 미만으로, 매 숙박마다 NFT를 발행하는 게이미피케이션 모델에 최적이다.

## 2. 세 가지 핵심 기능 (Three Pillars)

### 2.1. RWA (Real World Asset) -- 빈집 자산 토큰화

농촌 빈집의 소유권을 SPL Token으로 분할 발행하여, 글로벌 투자자가 소액으로 빈집 재생에 참여할 수 있도록 한다.

- **빈집 소유권의 분할 토큰화**: 1채의 빈집을 N개의 토큰으로 분할. 토큰 1개 = 감정가의 1/N
- **글로벌 투자자의 농촌 재생 참여**: USDC로 토큰을 매수하여 빈집 리모델링 자금 조달
- **임대 수익 자동 배당**: 숙박 수익이 발생하면 스마트 컨트랙트가 토큰 보유 비율에 따라 자동 분배
- **ESG/임팩트 투자 매력**: 농촌 재생이라는 사회적 가치 + 실물 자산 기반 수익

### 2.2. Gamification + NFT -- 숙박 경험 보상

기존 로드맵의 "Eco-Points / Rural Stamps"를 온체인으로 구현하여 검증 가능한 보상 시스템을 구축한다.

- **Experience Badge NFT**: 숙박 완료 시 해당 숙소의 한정판 NFT를 자동 발행. 숙소별, 계절별 고유 디자인
- **Eco-Points 온체인 포인트**: 숙박, 리뷰 작성, 체험 활동 참여로 포인트 적립. 할인 쿠폰, 특별 체험으로 교환
- **등급 시스템**: Bronze -> Silver -> Gold -> Platinum. 상위 등급일수록 할인율, 우선 예약, VIP 초대 혜택
- **컬렉션 가치**: 여러 지역의 Experience Badge를 수집하는 "한국 농촌 여행 컬렉터" 문화 형성

### 2.3. Blockchain Payment -- 암호화폐 결제

기존 Stripe/PayPal과 병행하여 SOL/USDC 결제 옵션을 추가한다.

- **SOL/USDC 결제**: Phantom, Solflare 등 지갑으로 직접 결제
- **기존 결제와 병행**: 게스트가 Card / Crypto 중 선택. 기존 Booking Flow에 탭 전환 UI 추가
- **에스크로 패턴**: 결제 대금을 스마트 컨트랙트가 보관하고, 체크인 완료 후 호스트에게 릴리스
- **글로벌 편의성**: 환전 없이 전 세계 어디서든 결제 가능, 수수료 최소화

## 3. 타겟 사용자 확장

### 3.1. Web3 네이티브 여행자
- 크립토 자산을 보유한 디지털 노마드
- 암호화폐로 직접 결제하고, NFT 보상을 받는 경험을 선호
- 한국 농촌의 독특한 체험 + Web3 요소의 조합에 관심

### 3.2. 임팩트 투자자
- RWA 토큰을 통해 한국 농촌 재생에 소액으로 참여
- 실물 자산 기반의 안정적 수익 + 사회적 가치(ESG)를 동시 추구
- 부동산 투자의 진입장벽(높은 자본, 지역 제한)을 블록체인으로 해소

### 3.3. NFT 컬렉터
- 독특한 한국 농촌 체험 NFT를 수집하는 문화적 가치에 관심
- 계절별, 지역별 한정판 배지를 모아 컬렉션 완성
- 커뮤니티 내 희소성 기반 사회적 자본 형성

## 4. 비전 정렬 (Vision Alignment)

### 4.1. 기존 미션과의 정합성
> "빈집의 재탄생, 연결의 부활" (기존 미션)

블록체인은 이 미션을 강화한다:
- **재탄생**: RWA 토큰화로 리모델링 자금을 글로벌하게 조달
- **연결**: NFT를 통해 게스트-숙소 간 지속적 관계 형성. 투자자-마을 간 경제적 연결

### 4.2. 투명성과 탈중앙화의 농촌 거버넌스 기여
- 토큰 홀더(투자자)가 마을 운영 결정에 참여하는 DAO 거버넌스 가능성 (Phase 4)
- 임대 수익, 리모델링 비용 등이 온체인에 공개되어 투명한 운영 보장

### 4.3. 기존 KPI 확장

| 기존 KPI | 블록체인 확장 KPI |
|----------|-----------------|
| 예약 수 | + 크립토 결제 비율 |
| 매출 | + 온체인 트랜잭션 볼륨 |
| 재방문율 | + NFT 보유자 재방문율 |
| - | RWA 토큰 홀더 수 |
| - | 총 NFT 민팅 수 |
| - | Eco-Points 활성 사용자 수 |
| - | RWA 배당 총액 |

## 5. 리스크 및 규제 고려사항

### 5.1. 한국 가상자산 규제
- **특정 금융정보법(특금법)**: 가상자산사업자(VASP) 신고 의무. 암호화폐 결제 수용 시 해당 여부 검토 필요
- **가상자산이용자보호법**: 2024년 시행. 이용자 자산 분리 보관, 보험 가입 의무
- **대응 전략**: 법률 자문을 통해 VASP 해당 여부 확인. 비해당 시 "결제 수단 제공자" 포지셔닝

### 5.2. RWA 관련 부동산 토큰화 법적 요건
- **자본시장법**: RWA 토큰이 "증권"에 해당할 경우 STO(Security Token Offering) 규제 적용
- **금융위원회 토큰 증권 가이드라인**: 분산원장 기반 증권의 발행/유통 규정
- **대응 전략**: SPV(특수목적회사) 설립, 증권신고서 제출, 인가 투자중개업자와의 협력

### 5.3. 사용자 지갑 분실/해킹 리스크
- 지갑 개인키 분실 시 자산 복구 불가
- **대응 전략**: Custodial 옵션 제공 (지갑 미연결 사용자를 위한 플랫폼 보관), 복구 가이드 안내
