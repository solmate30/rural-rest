# 09. RWA Pages UI Structure
> Created: 2026-02-20
> Last Updated: 2026-02-20

본 문서는 솔라나 기반의 빈집 소유권 공동투자 기능을 위해 설계된 RWA(Real World Asset) 전용 웹 페이지들의 아키텍처와 라우팅 구조를 정의합니다.

## 1. 아키텍처 원칙 (Separation of Concerns)

투자자(Investor)와 일반 숙박객(Guest)의 목적이 명확히 다르므로, 기존 예약 플로우(`/book`)와 혼재되지 않도록 독립된 `/invest` 라우트 하위로 모든 RWA 플로우를 격리합니다. 이를 통해 컴플라이언스(위험 고지, KYC 등) 요소가 일반 고객의 숙박 경험을 저해하지 않도록 합니다.

## 2. 주요 페이지 및 라우팅

### 2.1. RWA 탐색 허브 (`/invest`)
- **목적**: 토큰화된 빈집 자산들을 한눈에 보고 필터링하는 메인 투자 로비
- **주요 UI 요소**:
  - 현재 모집 중이거나 운영 중인 빈집 RWA 카드 리스트
  - 카드 내 정보: 토큰 가격 (USDC), 예상 연수익률(Yield), 모집률 프로그레스 바 (`Shadcn Progress` 컴포넌트)
  - **지역 검색 (Quick Chips)**: 칩 형태로 제공되는 지역('전주', '안동', '강릉' 등)을 눌러 빠르게 필터링
  - **고급 필터 (Filters 모달/패널)**:
    - **예상 수익률(APY) 필터링**: 연 8% 이상 수익이 예상되는 고수익률 물건 등 슬라이더 조절 기능
    - **토큰 가격대(Price) 필터링**: 1토큰당 5만 원 이하인 소액 투자 물건 등 예산 범위 설정
    - **펀딩 상태(Status) 필터링**: [진행 중(Active)], [오픈 예정(Coming Soon)], [마감된(Sold Out)] 상태별 모아보기
    - **숙소 테마 필터링**: "한옥", "오션뷰", "숲속 오두막" 등 특정 컨셉 모아보기

### 2.2. 빈집 RWA 상세 및 투자 페이지 (`/invest/:listingId`)
- **목적**: 특정 빈집에 투자를 결정하고 실제로 지갑을 통해 USDC 결제(서명)를 진행
- **주요 UI 요소**:
  - 상단: 숙소 갤러리 및 상세 정보 (리모델링 이력, 월 평균 예약률 등 객관적 지표 위주)
  - 우측 고정 패널 (Sticky): 토큰 요약 정보 (발행량, 시가총액, 연수익률)
  - 하단 차트: 과거 숙박 수익 기반의 배당금 이력 차트
  - **결제 폼**: 
    - 구매할 토큰 수량 입력 -> 총 USDC 자동 계산
    - "Connect Wallet to Invest" 또는 "Buy with USDC" 버튼
    - 하단에 투자 위험(원금 손실 등) 고지 Alert 박스 (`Shadcn Alert` 컴포넌트)

### 2.3. 내 투자 수익 대시보드 (`/my-investments`)
- **목적**: 사용자가 투자한 모든 빈집 RWA의 자산 가치 변동 및 배당금을 통합 관리
- **주요 UI 요소**:
  - Total Invested (총 투자금) 및 Current Value (현재 가치) KPI 위젯
  - 미수령 배당금(Pending Dividend) 총합 강조표시
  - 개별 투자 항목 카드: **"Claim Dividend (수익금 수령)"** 버튼 상시 노출 (온체인 트랜잭션 브릿지)

## 3. 재사용 가능한 UI 컴포넌트 (Shadcn 기반)
본 프로젝트의 기존 디자인 미학을 유지하기 위해 다음 컴포넌트들을 적극 재활용합니다.

- **`Card`**: 빈집 리스트 뷰 및 투자 상세의 정보 블록으로 사용. 모서리를 둥글게(radius) 처리하여 부드러운 느낌 제공.
- **`Badge`**: '모집 중', '배당 완료', '인기' 등의 상태값을 표시할 때 테두리 없는(outline) 뱃지로 활용.
- **`Button`**: 
  - 기본 숙박 버튼은 Warm Brown 계열 유지. 
  - 단, **지갑 연동 및 암호화폐 결제 버튼**은 사용자에게 동작의 차이를 무의식적으로 인지시키기 위해 별도의 브랜드 컬러(예: Primary Green 또는 보라색 톤)를 배정할 것을 권장.

## 4. 기존 페이지와의 연결 고리
일반 사용자가 숙소를 탐색하다가 RWA 투자로 자연스럽게 유입될 수 있도록 훅(Hook)을 추가합니다.

- **`/trips/:id` (일반 숙소 상세)**: 만약 해당 숙소가 RWA 토큰화된 숙소라면, 요금표 아래에 옅은 배너 알림을 띄웁니다.
  - "이 숙소는 Rural Rest를 통해 공동 소유(RWA)가 가능한 공간입니다. [수익 구조 알아보기 →]" 클릭 시 `/invest/:id`로 이동.
