## 4. Host 관점 -- 토큰화 신청 UI (`/admin/tokenize`)

### 4.1. 토큰화 신청 폼
```
┌─────────────────────────────────────┐
│ 내 숙소 토큰화 신청                  │
│                                     │
│ 숙소 선택:  [양평 돌담 고택 ▼]       │
│                                     │
│ 감정평가서: [파일 업로드]             │
│ 등기부등본: [파일 업로드]             │
│ 희망 토큰 수: [ 10,000 ]            │
│                                     │
│ ⚠️ 법률 검토 후 승인까지             │
│    2-4주 소요됩니다.                 │
│                                     │
│ [신청하기]                           │
└─────────────────────────────────────┘
```

### 4.2. 심사 상태 표시
```
신청 ──●── 법률 검토 ──○── 감정평가 ──○── 승인 ──○── 토큰 발행
       ↑ 현재 단계
```
- 진행 중 단계: Primary Green 원
- 미진행 단계: Gray 원
- 각 단계별 예상 소요 기간 및 메모 표시

## 5. 반응형 & 디자인 시스템 준수
- 투자 카드: `rounded-3xl`, `shadow-sm`, 호버 시 `shadow-md` 트랜지션
- 진행률 바: Primary Green 배경, `rounded-full`
- 수익률 양수: `text-green-600`, 음수: `text-red-600`
- 위험 안내: `bg-amber-50 border-amber-200 text-amber-800` 배경
- 차트: Recharts 또는 Chart.js (기존 프로젝트 의존성 확인 후 결정)
- 반응형: 데스크톱 2열, 모바일 1열 그리드
