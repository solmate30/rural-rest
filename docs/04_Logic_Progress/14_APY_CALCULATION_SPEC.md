# 14. APY 계산 명세

- 생성: 2026-03-28
- 관련 파일: `web/app/routes/api.rwa.save-mint.ts`

---

## 공식

```
연간 총 매출  = pricePerNight × 365 × 점유율
NOI          = 연간 총 매출 × (1 - 운영비율)
투자자 연수익  = NOI × 투자자 배분 비율
APY          = 투자자 연수익 / 감정가
```

코드:
```ts
const annualGross   = pricePerNight * 365 * OCCUPANCY_RATE;
const noi           = annualGross * (1 - OPERATING_COST_RATIO);
const investorReturn = noi * INVESTOR_SHARE;
return Math.round(investorReturn / valuationKrw * 10000); // basis points
```

---

## 상수

| 상수 | 값 | 근거 |
|------|----|------|
| `OCCUPANCY_RATE` | 0.55 | 농촌 단기임대 평균 점유율 55% 가정 |
| `OPERATING_COST_RATIO` | 0.45 | 운영비 45% (아래 세부 내역 참고) |
| `INVESTOR_SHARE` | 0.30 | SPV 수익 배분 구조 — 투자자 30% |

### 운영비 세부 내역 (총 매출 대비)

| 항목 | 비율 |
|------|------|
| 청소 / 관리 | 15~20% |
| 플랫폼 수수료 (에어비앤비 등) | 3~5% |
| 유지보수 / 수선 | 5~10% |
| 보험 / 세금 | 5~10% |
| 기타 | ~5% |
| **합계** | **약 35~50% → 45% 적용** |

---

## SPV 수익 배분 구조

```
NOI (순영업이익)
  ├── 지자체      40%  (오프체인 정산)
  ├── 마을운영자   30%  (온체인 USDC 지급)
  └── SPV 투자자  30%  (온체인 USDC 배당) ← APY 계산 기준
```

---

## 수치 예시

| 조건 | 값 |
|------|----|
| 숙박가 | ₩180,000/박 |
| 감정가 | ₩300,000,000 (3억) |
| 연간 총 매출 | 180,000 × 365 × 0.55 = ₩36,135,000 |
| NOI (운영비 45% 차감) | 36,135,000 × 0.55 = ₩19,874,250 |
| 투자자 연수익 (30%) | 19,874,250 × 0.30 = ₩5,962,275 |
| **APY** | **5,962,275 / 300,000,000 ≈ 1.99%** |

> 이전 공식(운영비 미반영)은 동일 조건에서 APY 3.6%로 약 1.8배 과대 추정.

---

## 한계 및 향후 개선

- 점유율(55%), 운영비율(45%)은 고정 상수 — 매물별 실적 데이터 축적 후 동적 반영 가능
- 현재는 추정치(projected)이며 실제 배당은 `rwa_dividends` 테이블 기준으로 지급
- Pyth oracle 연동 시 `KRW_PER_USDC` 환율도 실시간 반영 가능
