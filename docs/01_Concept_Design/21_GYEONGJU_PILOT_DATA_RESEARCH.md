# 경주 파일럿 프로젝트 실제 데이터 조사
> Created: 2026-03-16 00:00
> Last Updated: 2026-03-16 01:00

본 문서는 Rural Rest의 Phase 1 파일럿 지역인 **경주**의 실제 시장 데이터를 조사·정리한 것이다. **Mock 데이터 구체화를 위한 근거 자료**로 활용한다.

## ⚠️ 중요 전제 조건

본 문서의 모든 수치와 시나리오는 다음 가정을 전제로 합니다:

1. **20년 무상 사업권**: 지자체가 빈집 소유권 취득 + 리모델링 완료 후 Rural Rest에 20년 무상 운영권 제공 (**협상 완료 가정**)
2. **리모델링 완료 상태 인수**: Rural Rest는 완성된 숙소를 인수하며 리모델링 비용 부담 없음
3. **운영 수익**: 시장 평균 기반 추정으로, 실제 운영 결과와 다를 수 있음

**이 문서는 사업 계획서가 아닌 Mock 데이터 설계를 위한 시장 조사 자료입니다.**

## 0. Mock 데이터 전략 및 의사결정 과정

### 0.1. 파일럿 지역: 경주

- 빈집 가격 2,000만~4,000만원 (안동 1.9억 대비 4.8배 저렴)
- KTX 2시간 (외국인 배낭족 접근 용이)
- 황리단길 공실률 0%, 추석 방문객 70만명 → 안정적 수요
- **향후 전략**: Phase 1 경주 → Phase 2 안동 확장 (고급 라인)

### 0.2. 경주 5채 위치 선정 근거

**Q1: 왜 황리단길이 아닌 황오동/성건동인가?**
- **현실**: 황리단길 중심부는 공실률 0%, 신축 풀빌라 7~10억대로 전환
- **대안**: 황리단길 도보권 (황오동, 성건동) = 도보 8~12분 거리
- **근거**: 폐가정비사업 실제 철거 기록 (황오동 2023년, 성건동 2024년)
- **가격**: 3,000만~4,000만원 (농촌빈집은행 정읍 사례 2,500만원 대비)

**Q2: 왜 농촌빈집은행 데이터를 사용했는가?**
- **신뢰성**: 정부 공식 플랫폼 (농림축산식품부 운영)
- **실거래 사례**: 88건 매물, 정읍 2,500만원 실제 거래 (2024년)
- **한계**: 경주 불참 (경북은 예천만 참여) → 타 지역 사례 유추

**Q3: 5채 프로필은 어떻게 정했는가?**
- **황오동 (3000)**: 폐가정비사업 2023년 철거 지역, 황리단길 도보권
- **성건동 (3001)**: 100년 한옥 급매물 지역, 첨성대 인근
- **동천동 (3002)**: 2025년 도시재생 빈집 밀집 지역, 경주역 인근
- **건천읍 (3003)**: 2024년 폐가정비사업 철거 지역, 농촌
- **안강읍 (3004)**: 2023~2024년 2년 연속 철거 기록, 농촌

**결정 근거**: 실제 빈집 철거/정비 기록 기반 (가상 위치 아님)

### 5.2. 경주 구도심 빈집 매매가 추정

*   **경주 특수성**: 역사문화 미관지구 내 토지는 개발 제한 + 한옥 건축 의무로 시세 상승 제한적
*   **추정 범위**: 
    *   **소규모 한옥 (토지 50~100평, 건물 20평)**: 5,000만~1억원
    *   **중규모 한옥 (토지 200평, 건물 40평)**: 1.5억~3억원
    *   **대형 고택 (양동마을급)**: 3억~5억원 이상
*   **빈집 할인**: 방치된 빈집은 일반 시세 대비 30~50% 저렴 (수리 비용 감안)

### 5.3. Rural Rest 파일럿 5채 선정 이유

**빈집 시장 가격대 (참고):**
- **농촌빈집은행 (그린대로)**: 1,200만원 ~ 6억원 (88건 매물, 2025년)
- **실제 거래 사례**:
  - 전북 정읍: 토지 120평 + 건물 30평 = **2,500만원** (2024년)
  - 경남 거창: 단독주택 = 농촌빈집은행 첫 거래 (2025년 9월)
  - 경주 건천읍: 토지 160평 + 건물 30평 (신축) = **1억 2,000만원**
  - 안동 와룡면: 건물 107평 + 대지 353평 = **1억 9,000만원**
- **경북 부동산 시세**: 2년간 평균 27% 하락 (경산, 구미, 경주 등)

**경주 5채 위치 선정:**
- 황오동/성건동: 구도심 빈집, 황리단길 접근성
- 동천동: 경주역 인근, 도시재생사업 대상
- 건천읍/안강읍: 농촌 빈집, 불국사·남산 접근성

### 7.1. localhost://3000 - 황오동 청송재

    - 황리단길 카페거리: 도보 8분 (600m)

    - 경주역: 버스 10, 11번 → 황오동 정류장 하차 (20분)
